Phi Code Puzzle

Phi Code is a code developed directly for beginners.
This coding language enables beginners to seamlessly transfer to more modern coding language!








FAQ:


Phi φ is an ancient greek letter. In mathematics, it is the golden ratio. So, we aimed to create a coding language as beautiful as φ in nature.
